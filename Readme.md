# Installation

> ***Open the terminal and type following commands.***
```
$ curl https://raw.githubusercontent.com/TheDarkRoot/Terpack/master/Terpack.sh -o Terpack.sh
$ chmod +x TerPack.sh
$ ./TerPack.sh
```
